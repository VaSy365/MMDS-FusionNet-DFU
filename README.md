# MMDS-FusionNet: Multi-modal Deep Survival Fusion Network
Official Implementation for Journal Manuscript Submission
### Project Overview
This repository contains the official Python implementation for the research paper: "MMDS-FusionNet: A Multi-modal Deep Survival Fusion Network for Predicting Diabetic Foot Ulcer Outcomes"
### Key Features:
Architecture: Integration of DenseNet121 for image features and clinical metadata fusion.
Prognosis: DeepHit-based survival analysis for DFU risk prediction.
Explainability: Implementation of Grad-CAM for visual explanations and SHAP for clinical feature contribution analysis.
Performance: Achieved a Concordance Index (C-index) of 0.901 and Brier Score of 0.098.
